# ОБРСПИС (ОБРаботчик СПИСков)
Игрушечный лисп, написанный на C

## Linux install
Для Linux систем надо предварительно установить библиотеку `libedit`:
```bash
sudo apt install libeditline-dev
sudo apt-get install libedit-dev
```
